# АвтоДом — план развития сайта

**Позиционирование:** премиум-визуал (вдохновение Avilon Aura) + функциональность дилерской витрины (вдохновение Major Auto).
**Главная фишка:** честный калькулятор финальной цены — без «звоните, узнаете в салоне».

## Ключевые продуктовые выводы

- **Must-have у всех:** каталог с фильтрами, кредитный калькулятор, трейд-ин, тест-драйв, карта салонов, отзывы.
- **Боль рынка:** «цена по запросу». Решается фазой 4.
- **Точки роста на рынке:** видео, AR, прогноз остаточной стоимости, живой чат.
